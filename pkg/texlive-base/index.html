Package texlive-base

The package is not ported to Python 3 yet. You can visit Pagure for links to the upstream project. Ask for Python 3 status, and coordinate work with the upstream to port.

If upstream has Python 3 support but it is not yet in Fedora, get in touch with the Fedora maintainer (also visible in Pagure).


This package has a FESCo exception to stay on Python 2. See the TeX Live group.

See texlive-base in the Fedora Pagure, or go directly to the corresponding spec file.
